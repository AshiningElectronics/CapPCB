# CapPCB  

A capacitor PCB for the VESC  

Some capacitors that work from mouser are:  
647-UPW1J681MHD6  
647-UPW1J471MHD  
667-EEU-FC1J471L  

Read more about the VESC at:  
http://vedder.se/2015/01/vesc-open-source-esc/
